# PHASE1 Momentum - Research
> Date: 2026-04-27
> Status: Completed

## 목표
PHASE0에서 구축한 1,271종목 마스터 테이블을 대상으로,
4개 모멘텀 지표(DD, MACD, RSI, 52주 신고가)에 대한 백테스트를 수행하여
개별종목별 + 섹터별로 "뛰어난" 종목을 선별한다.

## 입력 데이터
- `PHASE0_classification/20260427_classification_master_verified.xlsx`
- 1,271종목, 18개 WICS tier1 섹터
- FDR로 일간/주간 OHLCV 수집

## 지표 설계

### 1. DD (Drawdown) — 5개 서브 지표
| 서브 지표 | 정의 | 용도 |
|-----------|------|------|
| Forward_MDD (무조건부) | 매 시점 t에서 향후 20영업일간 최대 낙폭 분포 (전체 기간) | 종목 간 하방 리스크 비교 |
| Forward_MDD (조건부) | 현재 DD 수준별(구간별) 향후 20영업일간 최대 낙폭 분포 | 현재 시점 진입 타이밍 판단 |
| Recovery Time | DD 이벤트 발생 후 전고점 복귀까지 걸린 영업일 수 (중앙값) | 회복 속도 평가 |
| Calmar Ratio | 연환산 수익률 / MDD | 수익 대비 리스크 종합 랭킹 |
| DD Velocity | DD 크기 / DD 도달 기간(일) | 급락 vs 완만하락 구분, 반등 패턴 예측 |

#### Forward_MDD 산출 로직
```
For each time t in [0, T-20]:
    window = P[t : t+20]
    running_max = cumulative_max(window)
    drawdown = (running_max - window) / running_max
    Forward_MDD(t) = max(drawdown)
```

#### 조건부 Forward_MDD 구간 정의
- 현재 DD 0~5% (고점 근처)
- 현재 DD 5~10%
- 현재 DD 10~20%
- 현재 DD 20%+ (깊은 조정)

### 2. MACD — 교차점 기반 (옵션 A: 비트코인 대화에서 설계한 로직)
- MACD Line: EMA(12) - EMA(26)
- Signal Line: EMA(9) of MACD Line
- Histogram: MACD - Signal
- **진입**: Histogram 골든크로스 (음→양 전환)
- **청산**: Histogram 데드크로스 (양→음 전환)
- **평가**: 전체 거래의 누적 수익률 (로그수익률, 왕복 수수료 2c 차감)
- 수수료: c = 0.1% (매수 0.05% + 매도 0.05% 근사, 증권사 평균)

### 3. RSI — 과매도 반등 전략
- RSI 기간: 14일 (Wilder 표준)
- **진입**: RSI가 30 하회 후 30 상향 돌파
- **청산**: RSI가 70 상회 후 70 하향 돌파
- **평가**: 전체 거래의 누적 수익률 (비용 반영)

### 4. 52주 신고가 — 브레이크아웃 전략
- **진입**: 52주(250영업일) 신고가 갱신 당일
- **청산**: 신고가 대비 -5% 하락 시 (트레일링 스톱)
- **평가**: 전체 거래의 누적 수익률 (비용 반영)

## 백테스트 설정

### 기간
- 1년 (최근 약 250영업일)
- 3년 (최근 약 750영업일)
- 5년 (최근 약 1,250영업일)
- 10년 (최근 약 2,500영업일)
- 데이터 부족 시: 존재하는 기간만 사용, 실제 분석 기간 명시

### 시간 프레임
- 일간 (Daily): 메인 분석
- 주간 (Weekly): 독립적 백테스트 (옵션 A)
- 각각 별도 Excel 시트로 출력

### 비용 모델
- 왕복 수수료: 2c = 0.2% (c = 0.1%)
- CR_adjusted = CR - 2c (로그수익률 프레임워크)

## 랭킹 및 선별 기준
1. **지표별 Top 20**: 각 지표의 누적 수익률 기준 상위 20종목
2. **섹터별 Top 3**: 18개 섹터 × 4개 지표 × Top 3
3. **복합 모멘텀**: 4개 지표 중 2개 이상에서 Top 50에 드는 종목
