# PHASE0 Classification - Research
> Date: 2026-04-27
> Status: Completed

## 목표
한국 주식(KOSPI+KOSDAQ) 전종목에 대한 섹터 분류 체계를 구축하고,
이후 Phase1~4 분석의 기반 데이터셋을 만든다.

## 보유 데이터

### 1. WICS 파일 (kospi_kosdaq_sector_classification.csv)
- 분류 체계: 3단계 (tier1 → tier2 → tier3)
- tier1 섹터: 19개 (특수목적법인 포함)
- 총 2,868종목, ticker 결측 44건
- 강점: 투자자 관점 분류 (2차전지/배터리, 방산/항공우주 등 테마 분리)
- 약점: ticker 결측으로 가격 데이터 매핑 불가 종목 존재

### 2. 상장법인목록 (상장법인목록 (2).xls)
- 분류 체계: KSIC (한국표준산업분류) 162개 업종
- 총 2,766종목
- 강점: 종목코드·상장일·지역 등 메타데이터 풍부
- 약점: 산업 통계용 분류라 투자 분석에 부적합 (이질적 종목이 동일 업종에 혼재)

## 확정된 의사결정

### 1. 섹터 분류 기준: WICS (tier1 18개)
- 특수목적법인(SPAC) 제외 → **18개 섹터**
- 사유: SPAC은 가격 움직임이 일반 주식과 상이, 모멘텀 백테스트 시 노이즈

### 2. 분석 단위: tier1 메인 + tier2 참조
- tier1(18개): 섹터별 모멘텀 비교의 메인 기준
- tier2: 참조용으로 같이 출력 (세부 drill-down 시 활용)
- 사유: tier2 단독 사용 시 섹터당 종목 수 부족으로 통계적 유의성 저하

### 3. 데이터 소스 및 조인 전략
- **마스터 리스트**: pykrx에서 전종목 리스트 직접 수집
- **섹터 태그**: WICS 파일의 tier1/tier2/tier3를 종목명 기준으로 조인
- 사유: pykrx가 가격 데이터 소스이므로 ticker 정합성 보장
- WICS ticker 결측 44건 문제를 자연스럽게 해결

### 4. 필터링 조건
- 시가총액: 1,000억 원 이상
- 일평균 거래대금: 10억 원 이상
- 사유: 소형주 유동성 부족 → MACD/RSI 시그널 신뢰도 저하, 실매매 어려움

### 5. 상장법인목록 활용
- 섹터 분류에는 사용하지 않음
- 상장일 정보만 추출하여 IPO 직후 종목 필터링에 보조 활용 가능
