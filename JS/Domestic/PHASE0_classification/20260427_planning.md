# PHASE0 Classification - Planning
> Date: 2026-04-27
> Status: Completed

## 구현 파이프라인

### Step 1: pykrx 마스터 리스트 수집
- `get_market_ticker_list(date, market)` → KOSPI/KOSDAQ 전종목 ticker
- 각 ticker에 대해 종목명 매핑
- 출력: DataFrame `{ticker, name, market}`

### Step 2: 시총 & 거래대금 필터링
- **시가총액**: `get_market_cap_by_ticker(date)` → 1,000억 원 이상
- **일평균 거래대금 (OR 조건)**:
  - 최근 20영업일 평균 거래대금 ≥ 10억 원 **OR**
  - 최근 60영업일 평균 거래대금 ≥ 10억 원
  - 둘 중 하나라도 충족하면 통과
- 산출: `get_market_trading_value_by_ticker()` 를 영업일별 호출 후 평균
- 사유: 20일은 최근 유동성 반영, 60일은 안정적 거래 종목 포착. OR 조건으로 포괄성 확보

### Step 3: WICS 섹터 조인
- WICS CSV 로드 → 특수목적법인 행 제거 (tier1 == '특수목적법인')
- **1차**: pykrx name ↔ WICS name 완전 일치(exact match) left join
- **2차**: 미분류 건수 확인 → 건수에 따라 fuzzy matching 또는 수동 검토 판단
- 우선주(예: "삼성전자우") 처리: WICS에 없을 가능성 높음 → 보통주의 섹터를 상속

### Step 4: 결과물 출력
- **Excel 파일** (`20260427_classification_master.xlsx`)
  - Sheet1 "전체종목": ticker, name, market, tier1, tier2, tier3, 시총, 일평균거래대금_20d, 일평균거래대금_60d
  - Sheet2 "섹터요약": tier1별 종목수, 평균시총, 총시총
  - Sheet3 "미분류": 섹터 매칭 안 된 종목 리스트
- **차트**: 섹터별 종목 수 bar chart

## 기술 스택
- Python 3.x
- pykrx: 종목 리스트, 시총, 거래대금
- pandas: 데이터 처리
- openpyxl: Excel 출력
- matplotlib: 차트
