## Step 1: `20260502_edgar_collector.py` (메인)

**역할**: SEC EDGAR에서 1,841 US 종목 × 10년치 Trailing 재무제표 수집.

### 입력
- `../PHASE0_classification/20260501_classification_master.xlsx`
  - **ADR 439종목 제외 → 1,841 US만**
  - 필요 컬럼: Symbol, Name, GICS_Sector, Country, Market Cap

### 처리
1. PHASE0 마스터에서 Country == 'United States' 인 1,841종목 로드
2. 각 종목에 대해 `Company(ticker).{income_statement|balance_sheet|cashflow_statement}(periods=10, period='annual', as_dataframe=True)` 호출
3. `edgar_field_map.py` 매핑으로 16개 핵심 필드 추출 (sector-aware)
4. 100종목마다 중간 캐시 저장 (중단 시 재개)

### 출력
- `cache/edgar_cache.pkl`
  ```python
  {
    'AAPL': {
      'income':   {'Revenue': {periods, values, label}, 'OperatingIncome': ...},
      'balance':  {'TotalAssets': {periods, values, label}, ...},
      'cashflow': {'OperatingCashFlow': ..., 'DividendsPaid': ..., 'StockRepurchase': ...},
      '_meta': {'sector': 'IT', 'is_financial': False, 'fetched_at': '2026-05-02', 'periods_received': 10}
    },
    ...
  }
  ```
- `logs/edgar_collection_log.csv`: ticker, status, n_fields_extracted, elapsed_sec, error

### 핵심 패턴
```python
from edgar import set_identity, Company
from edgar_field_map import extract_fields  # 모듈 분리

set_identity("Sang Il limited8090@gmail.com")

cache = load_existing('cache/edgar_cache.pkl')  # 증분 수집
for ticker in tickers_to_fetch:
    if ticker in cache and cache[ticker]['_meta']['fetched_at'] == TODAY:
        continue
    try:
        c = Company(ticker)
        is_financial = c.is_financial_institution
        income_df  = c.income_statement(periods=10, period='annual', as_dataframe=True)
        balance_df = c.balance_sheet(periods=10, period='annual', as_dataframe=True)
        cf_df      = c.cashflow_statement(periods=10, period='annual', as_dataframe=True)
        cache[ticker] = extract_fields(ticker, income_df, balance_df, cf_df, is_financial)
    except Exception as e:
        log_error(ticker, e)
    if i % 100 == 0:
        save_pickle(cache, 'cache/edgar_cache.pkl')
    time.sleep(0.1)   # SEC 10 req/sec 한도
```

### Rate limit & 재시도
- SEC 가이드라인: 10 req/sec → 0.1초 sleep
- 첫 호출 시 응답 자동 캐시 (`~/.edgar/`) — 재실행 빠름
- 429/Connection 에러 시 30초 대기 후 재시도 (최대 3회)

### 예상 소요
- **첫 실행**: 1,841 × 평균 8초 = 약 4시간 (캐시 없음)
- **이후**: 분기 보고 발표 시점에 30~50종목만 갱신 (5~10분)

**역할**: 두 캐시 통합 → 필터 → 카테고리 점수 → 곱셈 스코어 → 그룹·섹터 랭킹 → 출력.

#### Phase 0: 통합 DataFrame 구축
```python
df = pd.DataFrame(index=tickers)
df = merge(df, edgar_extracts)   # Trailing 재무
df = merge(df, yf_info_fields)   # Forward + 시총
df = merge(df, sector_master)    # GICS_Sector
```

#### Phase 1: 재무건전성 필터 — 다중 시ग널 + 섹터별 임계값 + NaN-safe
```python
# 섹터별 D/E 임계값
DE_THRESHOLD = {
    'Financials':  None,  # 면제
    'Real Estate': 8.0,
    'Utilities':   5.0,
    # 그 외 9섹터: 4.0 (default)
}
TIE_EXEMPT_SECTORS = {'Financials'}

def evaluate_filters(row):
    """반환: (단독 제외 사유 or None, [다중 시그널 위반 리스트])"""
    sector = row.get('GICS_Sector')
    de_cap = DE_THRESHOLD.get(sector, 4.0)

    # === 단독 제외 ===
    eq = row.get('edgar_StockholdersEquity')
    if pd.notna(eq) and eq <= 0:
        return ('자본잠식', [])
    mc = row.get('yf_marketCap')
    if pd.isna(mc) or (mc or 0) == 0:
        return ('시총누락', [])

    # === 다중 시그널 (2+ 위반시 제외) ===
    violations = []

    # 2. 고부채 (NaN-safe, 금융 면제)
    if de_cap is not None:
        ltd = row.get('edgar_LongTermDebt')
        if pd.notna(ltd) and pd.notna(eq) and eq > 0:
            if ltd / eq > de_cap:
                violations.append(f'고부채(D/E={ltd/eq:.1f})')

    # 3. 이자보상미달 (NaN-safe, 금융 면제)
    if sector not in TIE_EXEMPT_SECTORS:
        op = row.get('edgar_OperatingIncome')
        ie = row.get('edgar_InterestExpense')
        if pd.notna(op) and pd.notna(ie) and abs(ie) > 0:
            if op / abs(ie) < 1:
                violations.append(f'이자보상미달(TIE={op/abs(ie):.2f})')

    # 4. 3년 연속 영업적자 (NaN 면제)
    op_3y = row.get('edgar_OperatingIncome_3y') or []
    if len(op_3y) >= 3 and all(pd.notna(v) and v < 0 for v in op_3y[:3]):
        violations.append('3년영업적자')

    # 5. 3년 연속 영업CF 음수 (신규, NaN 면제)
    ocf_3y = row.get('edgar_OperatingCashFlow_3y') or []
    if len(ocf_3y) >= 3 and all(pd.notna(v) and v < 0 for v in ocf_3y[:3]):
        violations.append('3년OCF음수')

    return (None, violations)

# 적용
results = df.apply(evaluate_filters, axis=1)
df['solo_filter']   = results.apply(lambda x: x[0])
df['violations']    = results.apply(lambda x: x[1])
df['n_violations']  = df['violations'].apply(len)
df['passed_filter'] = df['solo_filter'].isna() & (df['n_violations'] < 2)
df['filter_reason'] = df.apply(lambda r:
    r['solo_filter'] if r['solo_filter']
    else (','.join(r['violations']) if not r['passed_filter'] else ''),
    axis=1)
df['data_completeness'] = df[['edgar_StockholdersEquity','edgar_LongTermDebt',
                              'edgar_OperatingIncome','edgar_InterestExpense',
                              'edgar_OperatingCashFlow']].notna().sum(axis=1)
```
필터 통과 종목만 스코어링. 제외 종목은 별도 시트로 보존(단독 제외 vs 다중 시그널 구분).

#### Phase 2: 카테고리 점수 (rank percentile)
```python
def rank_pct(s, ascending=True):
    return s.rank(ascending=ascending, pct=True)

# A. 밸류에이션 (낮을수록 좋음)
fwd_per_rank = rank_pct(df['forwardPE'], ascending=True)
peg_rank     = rank_pct(df['pegRatio'].clip(0, 10), ascending=True)
trailing_per_rank = rank_pct(df['trailingPE'], ascending=True)

def cat_valuation(row):
    if pd.notna(row['fwd_per_rank']):
        scores = [row['fwd_per_rank'], row.get('peg_rank')]
        return np.nanmean(scores)
    return row['trailing_per_rank']  # fallback
df['cat_valuation']  = df.apply(cat_valuation, axis=1)
df['val_is_forward'] = df['fwd_per_rank'].notna()

# B. 성장성 (높을수록 좋음)
fwd_growth_ranks = [rank_pct(df[c], ascending=False) for c in
                    ['revenueGrowth', 'earningsGrowth', 'fwd_eps_growth']]
trail_growth_ranks = [rank_pct(df[c], ascending=False) for c in
                      ['trailing_rev_yoy', 'trailing_op_yoy', 'trailing_eps_yoy']]

def cat_growth(row):
    fwd = [row[c] for c in fwd_growth_cols if pd.notna(row[c])]
    if fwd:
        return np.mean(fwd)
    trail = [row[c] for c in trail_growth_cols if pd.notna(row[c])]
    return np.mean(trail) if trail else np.nan
df['cat_growth']        = df.apply(cat_growth, axis=1)
df['growth_is_forward'] = (df[fwd_growth_cols].notna().any(axis=1))

# C. 주주환원 (부가지표, 메인 미반영)
df['shareholder_yield'] = (df['DividendsPaid'].abs() + df['StockRepurchase'].abs()) / df['marketCap']
df['return_to_op']      = (df['DividendsPaid'].abs() + df['StockRepurchase'].abs()) / df['OperatingIncome'].abs()
df['payout_ratio']      = df['DividendsPaid'].abs() / df['NetIncome']
df['buyback_yield']     = df['StockRepurchase'].abs() / df['marketCap']

# rank
sh_yield_rank   = rank_pct(df['shareholder_yield'], ascending=False)
return_to_op_r  = rank_pct(df['return_to_op'].clip(0, 1.5), ascending=False)
payout_rank     = rank_pct(df['payout_ratio'].clip(0, 1.0), ascending=False)
buyback_rank    = rank_pct(df['buyback_yield'], ascending=False)

df['cat_shareholder'] = pd.concat([sh_yield_rank, return_to_op_r, payout_rank, buyback_rank], axis=1).mean(axis=1)
```

#### Phase 3: 메인 스코어 (곱셈 + 페널티)
```python
FWD_FALLBACK_PENALTY = 0.50

def main_score(row):
    if pd.isna(row['cat_valuation']) or pd.isna(row['cat_growth']):
        return np.nan
    raw = row['cat_valuation'] * row['cat_growth']
    penalty = 1.0
    if not row['val_is_forward']:    penalty *= FWD_FALLBACK_PENALTY
    if not row['growth_is_forward']: penalty *= FWD_FALLBACK_PENALTY
    return raw * penalty
df['fundamental_score'] = df.apply(main_score, axis=1)
```

#### Phase 4: 그룹별 랭킹
```python
SIZE_LARGE = 10_000_000_000  # $10B
SIZE_MID   =  2_000_000_000  # $2B

def classify_size(mcap):
    if pd.isna(mcap) or mcap <= 0: return 'Small'
    if mcap >= SIZE_LARGE: return 'Large'
    if mcap >= SIZE_MID:   return 'Mid'
    return 'Small'

df['size_group']    = df['marketCap'].apply(classify_size)
df['size_rank']     = df.groupby('size_group')['fundamental_score'].rank(ascending=False, method='min')
df['shareholder_rank'] = df.groupby('size_group')['cat_shareholder'].rank(ascending=False, method='min')
df['sector_rank']   = df.groupby(['size_group', 'GICS_Sector'])['fundamental_score'].rank(ascending=False, method='min')
df['fundamental_rank_global'] = df['fundamental_score'].rank(ascending=False, method='min')
```

- `results/{날짜}_fundamental_ranking.xlsx`
  - 시트 `메인_Large` / `메인_Mid` / `메인_Small`: size_rank 순 정렬
  - 시트 `Top50_Global`: 전체 상위 50
  - 시트 `섹터별_Top10`: GICS 11섹터 × Top 10

본구현 후 다음 항목 100% 통과해야 production 진입:
