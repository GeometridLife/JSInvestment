#### 지표별 로직

**DD (5개 서브 지표)**
```
for each ticker:
    prices = cache[ticker]['Close']
    
    # Forward MDD (무조건부)
    for t in range(len(prices) - 20):
        window = prices[t:t+20]
        peak = window.cummax()
        dd = (peak - window) / peak
        forward_mdd[t] = dd.max()
    → 중앙값, 75th, 95th 퍼센타일 저장
    
    # Forward MDD (조건부)
    current_dd = (peak_to_date - prices) / peak_to_date
    bins: [0-5%, 5-10%, 10-20%, 20%+]
    → 각 bin별 forward_mdd 분포 저장
    
    # Recovery Time
    DD 이벤트 시작(전고점 이탈) → 전고점 복귀까지 영업일 카운트
    → 중앙값 저장
    
    # Calmar Ratio
    annualized_return = (prices[-1]/prices[0])^(252/len) - 1
    mdd = max drawdown of entire period
    calmar = annualized_return / mdd (mdd가 0이면 NaN)
    
    # DD Velocity
    각 DD 이벤트별: dd_size / dd_duration(days)
    → 평균 저장
```

**MACD**
```
for each ticker:
    ema12 = prices.ewm(span=12).mean()
    ema26 = prices.ewm(span=26).mean()
    macd_line = ema12 - ema26
    signal = macd_line.ewm(span=9).mean()
    hist = macd_line - signal
    
    # 골든크로스: hist[t-1] <= 0 and hist[t] > 0
    # 데드크로스: hist[t-1] >= 0 and hist[t] < 0
    
    # 각 거래쌍(GC→DC)의 로그수익률 산출
    CR = ln(P_exit / P_entry) - 2c
    
    → 승률, 평균수익률, 누적수익률, 거래횟수 저장
```

**RSI**
```
for each ticker:
    delta = prices.diff()
    gain = delta.clip(lower=0)
    loss = -delta.clip(upper=0)
    avg_gain = gain.ewm(span=14, adjust=False).mean()
    avg_loss = loss.ewm(span=14, adjust=False).mean()
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    
    # 진입: RSI가 30 하회 후 30 상향 돌파
    # 청산: RSI가 70 상회 후 70 하향 돌파
    
    → 승률, 평균수익률, 누적수익률, 거래횟수 저장
```

**52주 신고가**
```
for each ticker:
    high_250 = prices.rolling(250).max()
    
    # 진입: prices[t] >= high_250[t] (신고가 갱신)
    # 청산: prices[t] < entry_price * 0.95 (트레일링 스톱 -5%)
    # 실제로는 진입 후 고점 추적하며 고점 대비 -5%에서 청산
    
    → 승률, 평균수익률, 누적수익률, 거래횟수 저장
```
