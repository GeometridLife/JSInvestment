# 감사보고서 심층 분석 프로세스

> 본 문서는 비상장 기업 감사보고서를 분석할 때 사용한 프로세스를 정의한다. 마이크로인피니티 감사보고서 분석 사례를 기반으로 일반화된 워크플로우를 정리했다.

---

## 분석 철학

감사보고서 분석은 **단순 숫자 요약**이 아니라 **회사의 사업 모델·생애주기·전략적 위치를 숫자에서 역으로 추론**하는 작업이다. 따라서 다음 원칙을 따른다.

1. **숫자보다 맥락**: 절대값이 아닌 변화율과 구조 비율에 집중한다.
2. **재무제표 4종 + 주석의 균형**: 손익·재무상태·자본변동·현금흐름 + 주석을 모두 교차 검증한다.
3. **경영진이 강조하지 않는 것에 주목**: 충당금 신설, 평가손실, 우발채무는 종종 본문에 묻힌다.
4. **이익의 질(Quality of Earnings) 점검**: 회계 이익과 현금 이익의 갭이 핵심.
5. **2회차 검토 필수**: 1차 정리 후 반드시 놓친 부분 체크 → 수정.

---

## 4단계 프로세스

```
[1] 파일 리서치
    ↓
[2] 중요 내용 1차 정리
    ↓
[3] 놓친 부분 체크 (재검토)
    ↓
[4] 중요 내용 최종 정리 (수정·보완)
```

---

## Step 1 — 파일 리서치 (전체 통독)

### 목적
보고서 전체 구조와 회사의 전반적 윤곽을 파악한다. 이 단계에서는 분석하지 않고, **정보를 흡수**한다.

### 점검 항목

#### 1.1 감사보고서 메타정보
- [ ] 회사명, 결산기, 보고기간
- [ ] 감사인, 감사의견 (적정/한정/부적정/의견거절)
- [ ] 감사보고서 발행일
- [ ] 감사 시간 및 투입 인원 (외부감사 실시내용)
- [ ] 회계기준 (K-IFRS / K-GAAP / 중소기업회계처리 특례)

#### 1.2 회사 기본 프로필
- [ ] 설립일·본사 위치
- [ ] 주요 사업 (정관상 사업목적)
- [ ] 대표이사·주요 주주
- [ ] 자본금 및 발행주식수 (보통주/우선주 구분)
- [ ] 직원 수 (있는 경우)

#### 1.3 재무제표 4종 통독
- [ ] **재무상태표**: 자산·부채·자본의 절대 규모와 항목 구성
- [ ] **손익계산서**: 매출~당기순이익까지 흐름
- [ ] **자본변동표**: 자본 항목 변동 내역 (유상증자, 결손금 누적 등)
- [ ] **현금흐름표**: 영업·투자·재무 3대 흐름

#### 1.4 주석 통독
- [ ] 회계정책 (감가상각, 재고평가, 충당금 등)
- [ ] 특수관계자 거래 (반드시!)
- [ ] 우발채무·약정사항
- [ ] 자본 관련 사항 (RCPS, CB 등)
- [ ] 정부보조금
- [ ] 외화자산·부채

## Step 2 — 중요 내용 1차 정리

### 목적
원시 정보에서 **의사결정에 영향을 줄 수 있는 신호**를 추출한다.

### 핵심 분석 프레임워크

#### 2.1 손익 분석 — 변곡점 식별
```
Δ매출액 = 당기매출 - 전기매출
Δ매출총이익률 = 당기 GPM - 전기 GPM
영업이익 부호 변화 (적자→흑자 또는 그 반대)
```

**관전 포인트**:
- 매출 성장률이 50%를 초과하면 **변곡점** 가능성
- 매출총이익률 변화가 ±5%p 이상이면 **사업 구조 변화**
- 영업이익 흑자/적자 전환은 **반드시 단독 항목으로 강조**

#### 2.2 재무상태 분석 — 구조 비율
| 비율 | 계산식 | 의미 |
|------|--------|------|
| 부채비율 | 부채 / 자본 | 재무 건전성 |
| 유동비율 | 유동자산 / 유동부채 | 단기 지급능력 |
| 선수금/매출 | 선수금 / 연 매출 | 매출 가시성 |
| 재고/매출 | 재고 / 매출 | 운전자본 회전 |
| 결손금/자본금 | 결손금 / 자본금 | 자본 잠식 정도 |

#### 2.3 현금흐름 분석 — 이익의 질
```
현금전환비율 = 영업활동 현금흐름 / 당기순이익
```
- 1.0 이상: 이익이 현금으로 잘 전환됨
- 0.5~1.0: 양호
- 0~0.5: 운전자본 압박 의심
- **음수: 회계 이익은 흑자여도 현금 이익은 적자** → 주의 신호

#### 2.4 특수관계자 거래 분석 — 의존도
```
주거래처 매출 / 전체 매출 = 단일 거래처 의존도
```
- 80% 초과: 극심한 단일 고객 리스크
- 50~80%: 높은 의존도, 분산 노력 필요
- 30~50%: 일반적 핵심 고객 관계
- 30% 미만: 분산된 매출 구조

#### 2.5 자본 구조 분석 — IPO 단계 진단
- RCPS·CB 발행 이력 → 라운드별 발행가 변화 = **밸류업 트랙**
- 발행가 상승률이 라운드당 1.5배 이상이면 IPO 임박 시그널
- 결손금 vs 자본잉여금 → 자본 정리 필요성 평가

#### 2.6 우발채무·보증 분석
- 대표이사 개인보증 / 자본총계 → **회사-대표이사 분리도**
- 1.0 초과: 분리도 낮음 (중소기업 일반)
- 3.0 초과: 매우 위험, IPO 전 정리 필요

### 1차 정리 표준 출력 형식

```markdown
## 1차 정리

### 회사 개요

### 재무상태 — [핵심 메시지]
- 자산 구성
- 부채 구성
- 자본 구성

### 핵심 리스크

3.
```

## Step 3 — 놓친 부분 체크 (재검토)

### 목적
1차 정리에서 **간과한 항목**을 구조적으로 점검한다. 이 단계는 **반드시** 수행해야 하며, 보통 1차에서 30~40%의 정보를 놓친다.

### 재검토 체크리스트

#### 3.1 현금흐름표 디테일
- [ ] 영업현금흐름과 당기순이익의 차이 → 이익의 질
- [ ] 운전자본 변동 항목별 금액
- [ ] 투자활동의 CAPEX 종류별 분해
- [ ] 재무활동 0인지 여부 (자금조달 패턴)
- [ ] 비현금 거래 (전환사채 유동성 대체, 국고보조금 상계 등)

#### 3.2 주석 깊이 점검
- [ ] **충당부채** 신설 또는 급증 여부
- [ ] **재고자산 평가충당금** 변동
- [ ] **금융부채 만기 분석** (1년 이내 / 1-2년 / 2-5년 / 5년 이상)
- [ ] **외화자산** 종류 변화 (USD만 → 다국적 시그널)
- [ ] **보험가입** 종류 변화 (방산공제 등 산업 시그널)
- [ ] **연구개발비** 자본화 vs 비용화 비율

#### 3.3 인력·운영 신호
- [ ] 총 인건비 증가율 → 인력 확대 시그널
- [ ] R&D 인건비 증가율 → 차세대 제품 개발 강도
- [ ] 임차료 증가율 → 사업장 확장 시그널
- [ ] 퇴직급여 증가율 → 인력 누적 증가

#### 3.4 자본 구조 디테일
- [ ] RCPS 차수별 발행가·발행시기 → 밸류업 트랙
- [ ] RCPS 상환청구권 도래 시점
- [ ] CB 만기 도래 시점 → 다음 회계연도 자금 이벤트
- [ ] 우선주 배당률 및 누적 여부
- [ ] 전환가 조정조건 (refixing) 유무

#### 3.5 산업 맥락 매칭
- [ ] 업종 특수성 반영 여부 (방산·바이오·SW 등)
- [ ] 동종업계 비교 가능 항목
- [ ] 정부 R&D 보조금 의존도
- [ ] 특허·무형자산 가치 평가

#### 3.6 누락하기 쉬운 항목
- [ ] 임차보증금 변동 (사업장 확장)
- [ ] 임직원 퇴직연금 운용 형태 (DB/DC)
- [ ] 매도가능증권 구성
- [ ] 사용 제한된 예금 (담보·보증 목적)
- [ ] 대표이사·임원 보증 제공 내역
- [ ] 중소기업회계처리 특례 적용 여부

### 재검토에서 발견되는 전형적 누락
- "공사손실충당부채 19.8억원 신규 인식" (1차에서 그냥 부채 항목으로 지나칠 수 있음)
- "GBP 880,345.49 외화자산" (주석에 묻혀 있음)
- "방위산업공제조합 가입" (보험 변경의 산업적 의미)
- "RCPS 7차 2,250원/주 발행" (직전 라운드 1,000원 대비 2.25배 밸류업)
- "대표이사 서울보증보험 634억 보증" (단순 보증이 아닌 사업 규모의 척도)

## Step 4 — 중요 내용 최종 정리

### 목적
1차 정리에 3차에서 발견한 항목을 통합하여 **최종 분석 보고서**를 구성한다.

### 최종 보고서 표준 구조

```markdown
# 회사명 감사보고서 심층 분석

## 핵심 결론 (Executive Summary)
[2~3문장으로 회사 상태 요약]

## 1. 회사 개요
- 기본 정보 표
- 사업 정체성

## 2. 손익 분석
- 핵심 지표 표
- 흑자/적자 전환의 의미

## 3. 재무상태 분석
- 재무상태표 요약 표
- 핵심 관전 포인트 (3~5개)

## 4. 핵심 리스크 (예: 단일 거래처 의존도)
- 특수관계자 거래 분해
- 리스크 vs 강점 평가

## 5. 주주 구성
- 캡테이블
- IPO 단계 진단

## 6. 현금흐름 분석
- 3대 활동 요약
- 이익의 질 진단

## 7. 자금 부담 및 우발채무
- 대표이사 보증 현황
- 차기 자금 이벤트

## 8. 추가 운영 신호
- 인력·R&D·임차료
- 보험·외화·정부보조금

## 9. 종합 진단
- 🟢 Bull Case
- 🟡 Watch List
- 🔴 Risk Factors
- 한 줄 요약
- 산업 맥락

## 10. 분석 한계 및 추가 검토 권장 항목
```

### 시각화 활용 가이드

| 정보 유형 | 표현 방식 |
|---------|----------|
| 절대값·증감액 | 표 (당기/전기/증감/증감률 4열) |
| 시그널 분류 | Bull/Watch/Risk 컬러 박스 |
| 핵심 결론 | 박스로 강조 (배경색 + 두꺼운 테두리) |
| 비율·구성비 | 표 + 백분율 |
| 카테고리별 분류 | bullet list with bold prefix |

### 컬러 코드 표준
- **긍정 (Bull)**: `#00B050` (초록), 배경 `#E2EFDA`
- **주의 (Watch)**: `#ED7D31` (주황), 배경 `#FFF2CC`
- **위험 (Risk)**: `#C00000` (빨강), 배경 `#FCE4D6`
- **중립 (Header)**: `#1F4E78` (남색)
- **부 강조**: `#2E75B6` (밝은 파랑)

## 분석 시 자주 쓰는 표현 라이브러리

### 매출 변곡점 표현
- "매출 단 한 해 만에 N배 증가하고 영업이익이 △X억원에서 +Y억원으로 점프한 것은 명백한 변곡점이다"
- "흑자전환은 회복의 시작일 뿐 회복의 완료가 아니다 (결손금 잔존)"

### 단일 거래처 의존
- "[거래처]는 단순 거래처가 아니라 ① 매출의 X%를 차지하는 주 고객, ② Y%의 주주, ③ Z억원 전환사채의 인수자라는 삼중 관계로 깊이 결합되어 있다"
- "사실상 [거래처] 공급사슬의 종속 자회사에 가까운 구조"

### 운전자본 압박
- "회계상 이익 +X억원에도 불구하고 영업현금흐름 △Y억원 — 운전자본에서 Z억원이 빠져나간 결과"
- "양산 초기 단계 기업의 정상적 운전자본 확장으로 해석 가능하나, 지속 시 추가 자금 조달 필요"

### IPO 단계 진단
- "RCPS N차례 발행, 직전 라운드 대비 X배 밸류업 → IPO 임박 시그널"
- "결손금 잔존으로 RCPS 강제 상환 불가 → IPO 통한 보통주 전환이 사실상 출구 시나리오"

### 대표이사 보증
- "대표이사 개인 보증 약 X억원 — 자본총계 대비 N배"
- "IPO 성공 시 통상적으로 대표이사 개인 보증은 회사 보증으로 대체"

## 분석 시 피해야 할 함정

### ❌ 함정 1: 매출 증가 = 무조건 긍정
**올바른 시각**: 매출이 증가했어도
- 매출총이익률이 하락했는지
- 매출채권·재고가 동반 증가했는지
- 공사손실충당금이 신설되었는지
를 점검해야 한다.

### ❌ 함정 2: 회계 이익 흑자 = 사업 정상
**올바른 시각**: 영업현금흐름이 흑자인지가 더 중요하다. 회계 이익만 흑자이고 영업현금흐름이 적자라면 이익의 질이 낮다.

### ❌ 함정 3: 부채 증가 = 무조건 부정
**올바른 시각**: 부채 항목별 분해 필요. 선수금 증가는 향후 매출 가시성 (긍정), 차입금 증가는 현금 부담 (주의), 충당부채 증가는 미래 손실 인식 (부정).

### ❌ 함정 4: 우선주는 자본
**올바른 시각**: RCPS는 회계상 자본이지만 경제적 실질은 부채에 가깝다. 상환청구권이 도래하면 자금 부담이 발생할 수 있다.

### ❌ 함정 5: 감사 적정의견 = 회사 안정
**올바른 시각**: 적정의견은 "회계기준에 따라 작성됨"을 의미할 뿐, 사업 자체의 건전성을 보증하지 않는다.

본 프로세스를 적용하여 마이크로인피니티 제25기 감사보고서 분석에서 도출한 핵심 결론:

1. **변곡점 식별**: 매출 +149%, 영업이익 흑자전환 → 명백한 변곡점

6. **산업 시그널**: 방위산업공제조합 가입, 영국 GBP 자산 신규 등장 → 사업 확장 방향성
